# Skincare Brand Brochure Template

# Description

This project is a tri-fold brochure template designed for a skincare brand using Figma. The goal of this task was to create a clean, reusable, and print-ready brochure layout that can be easily customized for different brands.

# What I Did

Designed a tri-fold A4 brochure layout
Used a minimal and premium color palette to match a skincare brand aesthetic
Created clear content sections such as About Us, Contact Us, and Brand Message
Used placeholder text and image containers to make the design reusable as a template
Maintained proper visual hierarchy, spacing, and alignment for readability

# Design Approach

The brochure focuses on a soft, elegant, and natural look, suitable for skincare and beauty brands. Neutral background colors and clean typography were chosen to enhance clarity and provide a professional appearance.
